# MVP 범위

## 1차 MVP

백엔드 없이 프론트엔드만으로 동작하는 버전입니다.

### 포함

- YouTube 링크 입력
- YouTube videoId 추출
- 링크 리스트 렌더링
- 현재 재생 중인 영상 표시
- 영상 종료 시 다음 영상 자동 재생
- 마지막 영상 종료 시 첫 번째 영상으로 이동
- 전체 반복 ON 고정
- 리스트 아이템 삭제
- dnd-kit 기반 리스트 순서 변경
- `localStorage` 저장과 복원
- PWA 설치 가능한 manifest/service worker 설정

### 제외

- 백엔드
- 로그인
- DB 저장
- 공유 기능
- 여러 Loop List 관리
- YouTube 계정 권한 요청

## 우선순위

1. 재생 루프 동작
2. 링크 추가/삭제
3. `localStorage` 저장
4. 드래그 정렬
5. UI 다듬기
6. PWA 적용

## 완료 기준

- YouTube 링크 2개를 입력했을 때 `1번 -> 2번 -> 1번` 순서로 무한 반복된다.
- 새로고침해도 리스트가 유지된다.
- 리스트 순서를 바꾸면 바뀐 순서대로 재생된다.
- 마지막 영상이 끝나면 첫 번째 영상으로 돌아간다.
- Mac Chrome에서 PWA 설치가 가능하다.

## 1.5차 후보

- Space: 재생/일시정지
- N: 다음 영상
- P: 이전 영상
- L: 링크 입력창 포커스
- Delete: 선택한 아이템 삭제
