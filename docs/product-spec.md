# LoopTube 제품 명세

## 개요

LoopTube는 사용자가 YouTube 링크를 여러 개 등록하고, 등록한 영상을 순서대로 무한 반복 재생할 수 있는 웹/PWA 앱입니다.

일반 YouTube 플레이리스트가 많은 영상을 저장하고 탐색하는 데 초점을 둔다면, LoopTube는 지금 반복해서 듣고 싶은 영상 2~5개를 빠르게 구성하고 계속 재생하는 경험에 집중합니다.

## 제품 한 줄 설명

YouTube 링크를 모아 나만의 반복 재생 리스트를 만들고, 순서대로 무한 반복 재생하는 웹앱.

## 핵심 컨셉

- YouTube 링크 기반 반복 재생 리스트
- 로그인 전에는 로컬 저장
- 로그인 후에는 사용자 계정에 리스트 저장
- 웹 브라우저와 설치형 PWA 모두 지원
- Mac Chrome/Edge에서 데스크톱 앱처럼 설치 가능

## 주요 사용자 시나리오

### 링크 2개 반복

1. 사용자가 LoopTube에 접속한다.
2. YouTube 링크 2개를 입력한다.
3. 재생 버튼을 누른다.
4. 1번 영상이 재생된다.
5. 1번 영상이 끝나면 2번 영상으로 이동한다.
6. 2번 영상이 끝나면 다시 1번 영상으로 돌아간다.
7. 같은 흐름을 무한 반복한다.

### Mac에 앱처럼 설치

1. 사용자가 Chrome 또는 Edge에서 LoopTube에 접속한다.
2. 브라우저의 앱 설치 버튼을 누른다.
3. LoopTube가 독립 앱 창으로 실행된다.
4. Dock 또는 Launchpad에서 실행한다.

### 로그인 후 저장

1. 사용자가 Google 계정으로 로그인한다.
2. 기존 로컬 리스트가 서버 DB에 동기화된다.
3. 다른 기기에서 로그인한다.
4. 저장된 Loop List를 불러온다.

## 제약사항

- YouTube 영상은 YouTube IFrame Player API를 통해 재생한다.
- 삭제되었거나 비공개인 영상은 재생되지 않을 수 있다.
- 모바일/PWA 백그라운드 재생은 브라우저와 OS 정책을 따른다.
- 광고, 자동재생, 백그라운드 재생은 YouTube 정책과 브라우저 정책 영향을 받는다.
- LoopTube는 YouTube Premium 대체 서비스가 아니다.
